## 🇺🇸 English Version

### 🌟 Project Overview
ShineHub is a modern, responsive web application designed for car wash businesses. It provides a seamless experience for customers to explore services, create accounts, and book appointments online. The system features a bilingual interface (English/Arabic) and a robust backend for managing bookings and user authentication.

### 🚀 Key Features
*   **Bilingual Support:** Full English and Arabic localization with RTL support.
*   **Service Catalog:** Dynamic display of car wash packages with pricing and descriptions.
*   **Online Booking:** Simple multi-step form for scheduling appointments.
*   **User Authentication:** Secure login and registration system.
*   **Responsive Design:** Optimized for all devices (Desktop, Tablet, Mobile).
*   **Modern UI:** Clean glassmorphism design with smooth transitions.

### 🛠️ Tech Stack
*   **Frontend:** HTML5, CSS3 (Custom Variables & Flex/Grid), JavaScript (ES6+).
*   **Backend:** PHP (Object-Oriented approach for processing).
*   **Database:** MySQL (User and Booking management).

## 🇪🇬 النسخة العربية

### 🌟 نظرة عامة على المشروع
ShineHub هو تطبيق ويب حديث ومتجاوب مصمم لمغاسل السيارات. يوفر تجربة سلسة للعملاء لاستكشاف الخدمات، إنشاء الحسابات، وحجز المواعيد عبر الإنترنت. يتميز النظام بواجهة ثنائية اللغة (إنجليزية/عربية) وخلفية برمجية قوية لإدارة الحجوزات وتوثيق المستخدمين.

### 🚀 المميزات الرئيسية
*   **دعم لغتين:** تعريب كامل للغتين الإنجليزية والعربية مع دعم اتجاه النص من اليمين لليسار (RTL).
*   **كتالوج الخدمات:** عرض ديناميكي لباقات غسيل السيارات مع الأسعار والأوصاف.
*   **الحجز عبر الإنترنت:** نموذج بسيط من عدة خطوات لجدولة المواعيد.
*   **توثيق المستخدمين:** نظام تسجيل دخول وإنشاء حساب آمن.
*   **تصميم متجاوب:** محسن لجميع الأجهزة (كمبيوتر، تابلت، موبايل).
*   **واجهة حديثة:** تصميم زجاجي (Glassmorphism) نظيف مع انتقالات سلسة.

### 🛠️ التقنيات المستخدمة
*   **الواجهة الأمامية:** HTML5, CSS3, JavaScript (ES6+).
*   **الخلفية البرمجية:** PHP (لمعالجة البيانات).
*   **قاعدة البيانات:** MySQL (لإدارة المستخدمين والحجوزات).
